# whistle

## 简介
whistle是一款基于Node.js的跨平台web调试代理工具，主要用于抓包和调试HTTP/HTTPS请求。

## 主要功能
- 实时抓取HTTP/HTTPS请求
- 修改请求和响应
- 支持HTTPS抓包
- 支持自定义规则
- 支持插件扩展

## 安装方法
```bash
npm install -g whistle
```

## 启动方法
```bash
w2 start
```

## 常用命令
- 启动服务：
  ```bash
  w2 start
  ```
- 停止服务：
  ```bash
  w2 stop
  ```
- 重启服务：
  ```bash
  w2 restart
  ```
