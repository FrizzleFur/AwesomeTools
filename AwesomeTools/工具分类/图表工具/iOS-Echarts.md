# iOS-Echarts

## 简介
iOS-Echarts是一个基于ECharts的iOS图表绘制库，可以在iOS应用中快速集成各种图表。

## 主要功能
- 支持多种图表类型
- 高度可定制化
- 支持交互功能
- 与ECharts API兼容

## 安装方法
通过CocoaPods安装：
```ruby
pod 'iOS-Echarts'
```

## 使用示例
```swift
let chartView = PieChartView()
chartView.setOption(pieChartOption)
self.view.addSubview(chartView)
```
